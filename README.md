##  Workshop Objective
This workshop demonstrates how to implement core **Probabilistic Language Models** using a small document corpus. The goal is to build a modular NLP pipeline and explore **unigram, bigram, trigram, and backoff** language models using Python and NLTK, along with preprocessing and inverted indexing.
